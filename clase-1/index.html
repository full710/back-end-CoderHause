Clase 1 Back End

Tema 1 - Plantillas Literales

Las plantillas literales (o template literals) en JavaScript son una forma de trabajar con cadenas de texto más flexible y expresiva. Se definen usando comillas invertidas (`) en lugar de comillas simples (') o dobles (").

Tema 2 - Funciones

En programación, una función es un bloque de código reutilizable que realiza una tarea específica. Se define una vez y puede ser llamada (ejecutada) varias veces desde diferentes partes del programa.

Tema 3 - Scope

En programación, el scope (o alcance) se refiere al contexto donde las variables, funciones y objetos son accesibles. Determina desde dónde puedes usar o modificar una variable en tu código.

Tema 4 - Closures

Los cierres o clausuras es un concepto que se refiere a la capacidad de una funcion anidada de acceder a las variables de su funcion padre, incluso cuando la funcion padre ya termino de ejecutarse. Un closure en JavaScript es una función que "recuerda" el scope en el que fue creada, incluso después de que dicho scope haya terminado su ejecución. Esto permite que una función interna acceda a las variables de la función externa, incluso después de que esta última haya finalizado.

Tema 5 - Clases y POO

La Programación Orientada a Objetos (POO) es un paradigma de programación que organiza el código en torno a objetos, los cuales son entidades que combinan datos (atributos) y comportamientos (métodos).

Clase: Es un modelo o plantilla para crear objetos. Define las propiedades (atributos) y comportamientos (métodos) que tendrán los objetos.

Objeto: Es una instancia de una clase. Representa una entidad con atributos y métodos concretos.

Atributos: Son las propiedades o características del objeto. Métodos: Son las funciones que definen el comportamiento de los objetos.
