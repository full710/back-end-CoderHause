Bienvenidos a la clase 2 de Backend

Desestructuración

La desestructuración en programación es una técnica que permite extraer valores de arreglos o propiedades de objetos y asignarlos a variables de manera concisa. Es especialmente útil para simplificar el acceso a datos.

Valores por defecto

Los valores por defecto en programación son valores asignados automáticamente a variables o parámetros cuando no se les proporciona uno explícito.

Trabajo por modulos

Un módulo en programación es una unidad de código independiente y reutilizable que agrupa funciones, variables, u objetos relacionados. Se utiliza para organizar y estructurar programas grandes, promoviendo la reutilización y la mantenibilidad.

Clases y Objetos

Clases: Son plantillas o moldes que definen las propiedades (atributos) y comportamientos (métodos) que tendrán los objetos. Representan conceptos generales.

Objetos: Son instancias de una clase. Representan entidades específicas que se crean a partir de una clase y poseen valores concretos para sus atributos.

Metodos

Un método es una función asociada a un objeto o clase que define un comportamiento específico. Es una forma de realizar acciones relacionadas con ese objeto o de interactuar con sus propiedades.

Herencia

La herencia es un principio de la programación orientada a objetos (POO) que permite a una clase (llamada subclase o clase hija) adquirir propiedades y métodos de otra clase (llamada superclase o clase padre). Esto permite reutilizar y extender el comportamiento de clases existentes.

Variables y metodos estaticos

En programación orientada a objetos, variables y métodos estáticos pertenecen a la clase, no a las instancias de la clase. Esto significa que se acceden directamente a través de la clase sin necesidad de crear un objeto.

Includes

El método includes en JavaScript se utiliza para verificar si un arreglo o una cadena de texto contiene un elemento o una subcadena específica. Devuelve un valor booleano (true o false).

Async / Await

async y await son palabras clave en JavaScript que facilitan el manejo de código asíncrono, haciéndolo más legible y sencillo al trabajar con Promesas.

Object.values

El método Object.values en JavaScript devuelve un arreglo con los valores de las propiedades enumerables de un objeto. Es útil para trabajar directamente con los valores sin preocuparse por sus claves.

Object.entries

El método Object.entries en JavaScript devuelve un arreglo de pares clave-valor de las propiedades enumerables de un objeto. Cada par clave-valor se representa como un arreglo de dos elementos: [clave, valor].

Object.keys

El método Object.keys en JavaScript devuelve un arreglo con las claves (o nombres de las propiedades) de las propiedades enumerables de un objeto.
